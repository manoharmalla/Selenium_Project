# Selenium Hybrid Automation Framework

## Overview

This project is a Selenium Hybrid Automation Framework developed using Java. The framework combines the advantages of the Page Object Model (POM), Data-Driven Testing, and TestNG to create a scalable, maintainable, and reusable test automation solution.

The framework is designed to automate web application testing by separating test logic, page objects, test data, and utility classes.

---

## Tech Stack

- Programming Language: Java
- Automation Tool: Selenium WebDriver
- Test Framework: TestNG
- Design Pattern: Page Object Model (POM)
- Build Tool: Gradle
- Data-Driven Testing: Properties File
- IDE: IntelliJ IDEA

---

## Framework Features

- Hybrid Automation Framework
- Page Object Model (POM) Design Pattern
- Data-Driven Testing
- TestNG Test Execution
- Cross-browser support (if implemented)
- Reusable utility methods
- Configurable through Properties file
- Screenshot capture on test failure
- Detailed TestNG Reports
- Easy maintenance and scalability
